# Faraday rotation from ellipsometry data
This is a repository of scripts that parse ellipsometry data from .txt files, calculate Faraday rotation three different ways, and returns the values in a table and plot formatted for an excel file.

## Samples 
This script analyses data from the samples data folder. The calculated Faraday rotation of different samples is plotted with error bars. 

## Angles 
This script analyses data from the angle data folder. The calculated Faraday rotation of a thin-film of 1Sn15Au nanoparticles for different angles of incidence of 532 nm light is plotted with error bars.

## Si Control
This script analyses data from the samples data folder.
